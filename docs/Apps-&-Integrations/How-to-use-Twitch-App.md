<p>Twitch have grown to more than just a game streaming platform, you now can find music channels, talk shows, computer programming class, etc. on this platform.</p>
<p>You can put your favorite Twitch channels on Digital Signs with OptiSigns' Twitch app.<br>You can show or hide chat bar with your channel as well. Here's an example how it would look with chat bar on:</p>

<p class="rich-content-viewer_elementSpacing__208Ie"><u>Name</u>: Name of your Twitch asset, this is the name in your asset list. It will <u><strong>not</strong></u> be displayed on your screens.</p>
</li>
<li>
<u>Twitch URL</u>: Copy and paste the link to the Twitch channel you want to use here. For example: <a href="https://www.twitch.tv/shroud">https://www.twitch.tv/shroud</a> use the preview link to ensure you got the correct link.</li>
<li class="rich-content-viewer_unorderedList__1BJwx rich-content-viewer_elementSpacing__208Ie _3_7DB AvMd_ _310Mz rich-content-viewer_left__2p1aK _158eo _3_7DB">
<u>Theme:</u> Select Light or Dark Theme. This mainly only affect chat box</li>
<li class="rich-content-viewer_unorderedList__1BJwx rich-content-viewer_elementSpacing__208Ie _3_7DB AvMd_ _310Mz rich-content-viewer_left__2p1aK _158eo _3_7DB">
<p class="rich-content-viewer_elementSpacing__208Ie"><u>Chat box</u>: Chat box is important element of Twitch, because streamers usually interact with their viewers here. Check this box if you want to show chat. If not the streaming video would be full screen.</p>
</li>
</ul>

<p class="rich-content-viewer_text__XzvDs rich-content-viewer_elementSpacing__208Ie _3_7DB blog-post-text-font blog-post-text-color rich-content-viewer_left__2p1aK _158eo _3_7DB">You have created your Twitch app.<br>You can change the wall any time by click on it in the Files/Assets tab. </p>
<p class="rich-content-viewer_text__XzvDs rich-content-viewer_elementSpacing__208Ie _3_7DB blog-post-text-font blog-post-text-color rich-content-viewer_left__2p1aK _158eo _3_7DB">You can assign the newly created instance to your screen by going to Screens, click Edit screens and assign the wall to screens that you want.</p>
