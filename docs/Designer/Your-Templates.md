<p>In Designer, the <strong>Your Temp</strong> (short for “Your Templates”) tab allows you to save and reuse your created templates across multiple designs.</p>
<p>For more general information on the Designer app, see our<a href="https://support.optisigns.com/hc/en-us/articles/42087942047379-Getting-Started-with-Designer"> </a><a href="https://support.optisigns.com/hc/en-us/articles/42087942047379-Getting-Started-with-Designer" target="_blank" rel="noopener noreferrer"><strong><span class="wysiwyg-underline">Getting Started in Designer</span></strong></a><strong> </strong>guide.</p>
<p>This feature can be accessed from the Side Menu within Designer:</p>
<p><img src="https://support.optisigns.com/hc/article_attachments/42830684533779" width="73" height="596"></p>
<p>Within this tab, there are two options: <strong>Templates </strong>and <strong>Components.</strong></p>
<ul>
<li>
<strong>Templates </strong>allows you to save the current design as a Custom Template for use in later designs. This saves the <em><strong>entire design</strong></em> as a template.</li>
<li>
<strong>Components </strong>allows you to save a selected component (be that text, image, shape, widget, etc.) as a custom component for use in later designs. This saves <em><strong>just the selected component</strong></em> as a template.</li>
</ul>
<p>Below, we’ll cover how to save and store these custom templates and give examples of how they can be used.</p>
<hr><p><a name="Templates"></a></p>
<h2 id="h_01JZNYRE6VJ409YKBT820MMJ7S">Saving and Using Custom Templates</h2>
<p>The <strong>Templates </strong>tab has one option when you first open it: <strong>Add Current Design as Template</strong>. Nothing needs to be selected in order to use this.</p>
<p><img src="https://support.optisigns.com/hc/article_attachments/42830652782483" width="254" height="179"></p>
<p>Clicking this will bring a popup asking you to confirm your choice. Click <strong>Add</strong>. This saves whatever you have on the current design as a Custom Template, shared across your team.</p>
<p><img src="https://support.optisigns.com/hc/article_attachments/42830684537875" width="250" height="325" alt="custom template added"></p>

<tr><td style="border-style: solid; border-width: 1px;">The Custom Template may appear as a black box when it is first added to the design. However, it<strong> </strong>will still function normally. Saving the design and relaunching it should make it appear. </td></tr>
</tbody></table></figure>
<p>To use this template, simply click on it from the Side Menu:</p>
<p><img src="https://support.optisigns.com/hc/article_attachments/42830684540179" width="624" height="311" alt="custom template being added"><br>This will be present across any Designer instance.</p>

<h2 id="h_01JZNYRE6YMJ7DDHDK6W7DYTDV">Saving and Using Custom Components</h2>
<p>The <strong>Components </strong>tab has one option when you first open it: <strong>Add Current Selection as Component</strong>. A component on the current design needs to be selected for this to be used.</p>
<p><img src="https://support.optisigns.com/hc/article_attachments/42830652789907" width="250" height="170" alt="component tab add current selection as component button"></p>
<p>Clicking this will bring up a popup asking you to confirm your choice. Click <strong>Add</strong>. This saves the selected component as a Custom Component, shared across your team.</p>
<p><img src="https://support.optisigns.com/hc/article_attachments/42830652796563" width="257" height="253" alt="custom component added"></p>
