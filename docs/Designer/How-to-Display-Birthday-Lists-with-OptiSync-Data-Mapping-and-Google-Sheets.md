<p>If none of the Repeater designs resonate with the design you are trying to accomplish, you can create your own design within the repeater itself from scratch. Here's how:</p>
<ol>
<li>Open the<span> </span><strong style="box-sizing: inherit;" data-stringify-type="bold">Designer app</strong>
</li>
<li>Go to<span> </span><strong style="box-sizing: inherit;" data-stringify-type="bold">DataSource</strong><span> </span>→ Select your custom DataSource → Expand "<strong style="box-sizing: inherit;" data-stringify-type="bold">Row 1</strong>".</li>
<li>

<li>Once the customization is complete press "<strong>Update to Save</strong>".

<h2 id="h_01J182D7B4HZ8ZXSKD9BJNTCV7">How to Handle Empty Data Values in a Repeater</h2>
<p>Let's say you're playing this design as a single asset or running it like a slideshow, but it is nobody's birthday. You can set how you would like your Repeater to handle empty data values. </p>
<ol>
<li>In Designer, <strong>select your Repeater</strong> so it is highlighted.</li>
<li>Select <strong>Settings</strong> from the top bar.</li>
<li>In the far-right menu, use the drop-down menu for <strong>Empty Data Handling</strong> to select:
<ul>
<li>
<strong>Skip:</strong> Skip it, or get rid of additional repeater items if there isn't enough data to reach the set "Total Items displayed per Page".</li>
<li>
<strong>Use Default Value:</strong> Show default content, which is what your Repeater element looks like by itself.</li>
<li>
<strong>Use Blank:</strong> The Repeater will show nothing.</li>
</ul>
</li>
<li>Make sure to <strong>save</strong> your design.</li>
</ol>

<h2 id="h_01JB9TBZPY8P61ZDGDD2KVDFY0">Set Up Auto Sorting in Your Sheet </h2>
<h4 id="h_01JAZSQD73KK88CTKJDWY9FPQY">When handling large quantities of data in your spreadsheet, the ability to auto sort is incredibly useful and at times, essential. Instead of manually rearranging your data every time you add or update entries, the <span class="wysiwyg-color-green130">SORT</span> function does it for you automatically. </h4>
<table style="border-collapse: collapse; width: 100%; height: 44px;" border="1">
<tbody>
<tr style="height: 44px;">
<td style="width: 100%; height: 44px;">
<strong>Note:</strong> If you're using the example spreadsheet we provided above, there are specific instructions in that Google Sheet on how to set up auto-sorting for that data set-up. Please read the "<strong>Tips</strong>" section. If you're not using that spreadsheet, you can follow the example below to help you understand sorting. </td>

<h4 id="h_01JBABRV6MPTXK0QKV8QXTXW41"><strong>Example: Auto-Sorting a Birthday List</strong></h4>
<p>Imagine you have a list of employees and their birthdates. You want to keep this list sorted by the upcoming birthdays, and if two employees share the same birthdate, you want to sort their names alphabetically.</p>
<p>Let’s go over the <strong>Step-by-Step Instructions:</strong></p>
<p><strong>Step 1</strong>: Open Your Google Sheet. For simplicity, here’s how your data might look:</p>
<div>
<table style="width: 100%;">
<colgroup>
<col>

</tr>
<tr>
<td>Alice</td>
<td>12/05/1990</td>
</tr>
<tr>
<td>Bob</td>
<td>12/15/2001</td>
</tr>
<tr>
<td>Carol</td>
<td>12/05/1990</td>
</tr>

<p id="h_01JBABRV6MEBTE4BRZEHD5E5AX"><strong>Step 2: </strong>Select an Empty Cell</p>
<p>Click on an empty cell where you want the sorted list to appear. For this example, we’ll use cell A1 in a new tab.</p>
<p><strong>Step 3</strong>: Enter the <span style="color: #188038;">SORT</span> Formula</p>
<p>In the selected cell A1, enter the following formula and press Enter.</p>
<pre><span style="color: #188038;">={"Celebrant Name", "Birthdate"; SORT(A2:B, B2:B, true, A2:A, true)}</span></pre>
<p>The formula will immediately sort the data and display the results starting from Cell A1.</p>
<h3 id="h_01JBABRV6MTDZ7X39S1NXPRE9R">How the Formula Works:</h3>
<ol>
<li>
<strong><span style="color: #188038;">={"Celebrant Name", "Birthdate";</span></strong>
<ul>
<li>
<strong>Purpose</strong>: This creates the headers for your sorted data.</li>
<li>
<strong>Explanation</strong>: It sets up the column titles "<span class="wysiwyg-color-green130"><strong>Celebrant Name</strong></span>" and "<span class="wysiwyg-color-green130"><strong>Birthdate</strong></span>" at the top of the sorted list. The <strong>semicolon <span style="color: #188038;">;</span></strong> tells Google Sheets to display the sorted data below these headers.</li>
</ul>
</li>
<li>
<strong><span style="color: #188038;">SORT(A2:B, B2:B, true, A2:A, true)</span></strong>
<ul>
<li>
<strong><span style="color: #188038;">A2:B</span></strong>: This is the range of data you want to sort (including both columns: Celebrant Name and Birthdate).</li>
<li>
<strong><span style="color: #188038;">B2:B</span></strong>: This specifies the first column to sort by, which is Birthdate.
<ol>
<li>
<strong><span style="color: #188038;">true</span></strong>: This indicates that the sorting should be in <em><strong>ascending</strong> </em>order.</li>
<li>
<strong><span style="color: #188038;">false</span></strong>: This indicates that the sorting should be in <em><strong>descending</strong> </em>order. </li>
</ol>
</li>
<li>
<strong><span style="color: #188038;">A2:A</span></strong>: This specifies the second column to sort by, which is Celebrant Name.
<ol>
<li>
<strong><span style="color: #188038;">true</span></strong>: This indicates that names should also be sorted in <em><strong>ascending</strong> </em>order.</li>
<li>
<strong><span style="color: #188038;">false</span></strong>: This indicates that names should also be sorted in <em><strong>descending</strong> </em>order.</li>
</ol>
</li>
</ul>
</li>
</ol>
