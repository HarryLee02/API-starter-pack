<h2 id="h_01JY4FP78M8AFTRR35N9KEGWT5">Adding Shapes and Elements</h2>
<p>The <strong>Shapes </strong>option allows you to add premade shapes or elements to the design.</p>
<p><img src="https://support.optisigns.com/hc/article_attachments/42307250831507" width="624" height="45"></p>
<p>Choosing <strong>Shapes </strong>from the Top Menu will open the <strong>Elements </strong>tab on the Side Menu:</p>
<p><img src="https://support.optisigns.com/hc/article_attachments/42307234464275" width="302" height="560"></p>
<p>Here, you can choose between <strong>Shapes </strong>and <strong>Elements </strong>to add to the design. There are a vast number of these, so pick ones which suit your needs.</p>
<p>Clicking a Shape or Element will add it to the design:</p>
<p><img src="https://support.optisigns.com/hc/article_attachments/42307250831891" width="624" height="373"></p>
<p>Shapes are reactive and can be resized or rotated at will:</p>
<figure class="wysiwyg-image"><img style="aspect-ratio: 259/255;" src="https://support.optisigns.com/hc/article_attachments/42307234464787" alt="firefox_9GYjGYVRec.gif" width="259" height="255"></figure>

<h2 id="h_01JY4FP78TQP0VZ3DKFTHT2RGP">Shape and Element Options</h2>
<p>The Shape or Element itself has a full set of options:</p>
<p><img src="https://support.optisigns.com/hc/article_attachments/42307234465299" width="449" height="573"></p>
<ul>
<li>
<strong>Duplicate - </strong>Creates an exact duplicate of the selected shape or element</li>
<li>
<strong>Delete </strong>- Removes the selected shape or element from the design</li>
<li>
<strong>More</strong><ul>
<li>
<strong>Copy </strong>- Copies the selected shape or element to be Pasted later</li>
<li>
<strong>Copy Style </strong>- Copies the style of the selected shape or element to transfer to another shape or element</li>
<li>
<strong>Paste </strong>- Pastes element from clipboard over selected shape or element</li>
<li>
<strong>Duplicate </strong>- Creates an exact duplicate of the selected shape or element</li>
<li>
<strong>Add Component </strong>-</li>
<li>
<strong>Delete </strong>- Removes the selected shape or element from the design</li>
<li>
<strong>Layer </strong>- Options to change the selected shape or element layer<ul>
<li>
<strong>Bring Forward </strong>- Brings selected shape or element up one layer</li>
<li>
<strong>Bring to Front </strong>- Brings selected shape or element to the top layer</li>
<li>
<strong>Bring Backward </strong>- Brings selected shape or element down one layer</li>
<li>
<strong>Bring to Back </strong>- Brings selected shape or element to the bottom layer</li>
</ul>
</li>
<li>
<strong>Align Elements </strong>- Options to change the selected shape or element alignment<ul>
<li><strong>Left</strong></li>
<li><strong>Center</strong></li>
<li><strong>Right</strong></li>
<li><strong>Top</strong></li>
<li><strong>Middle</strong></li>
<li><strong>Bottom</strong></li>
</ul>
</li>
<li>
<strong>Link </strong>- Add a URL to the selected shape or element.</li>
<li>
<strong>Animate </strong>- Add an animation to the selected shape or element.</li>
</ul>
</li>
</ul>
<p>Clicking the Shape or Element will change the Top Menu and apply directly to the selection:</p>

<p><a name="Link"></a></p>
<h3 id="h_01JY4FP79BTF31P8124YBYB1YC"><span style="color: #434343;">Link</span></h3>
<p><img src="https://support.optisigns.com/hc/article_attachments/42307234471443" width="73" height="31"></p>
<p>Selecting the <strong>Link </strong>option will allows you to turn your shape or element into a hyperlink. It will open the <strong>Link Options </strong>section on the Side Menu:</p>
<p><img src="https://support.optisigns.com/hc/article_attachments/42307234471699" width="337" height="147"></p>
<p>Clicking <strong>Add Link </strong>will prompt you to enter a URL or select an asset:</p>
<p><img src="https://support.optisigns.com/hc/article_attachments/42307250841619" width="624" height="313"></p>
<p>Once a link or asset has been chosen, more options will appear in the Link Options section:</p>
<p><img src="https://support.optisigns.com/hc/article_attachments/42307250842003" width="325" height="353"></p>
<p>Expanding the <strong>Scale </strong>section will provide this option for your asset:</p>
<p><img src="https://support.optisigns.com/hc/article_attachments/42307234474131" width="335" height="113"></p>
<p>Here, choose how the image will be scaled. There are three options:</p>
<ul>
<li>
<strong>Fit (default) </strong>- Fits the image to the screen.</li>
<li>
<strong>Stretch </strong>- Stretches the image to fill the entire screen.</li>
<li>
<strong>Zoom </strong>- Zooms in on the image to fill up the entire screen. Not recommended for small images on large displays, as this will create blurriness.</li>
</ul>
<figure style="width: 100%;" class="wysiwyg-table"><table style="border-style: solid; border-width: 1px;"><tbody>
<tr><td style="border-style: solid; border-width: 1px;"><strong>NOTE</strong></td></tr>
<tr><td style="border-style: solid; border-width: 1px;">If a URL is provided, the <strong>Scale </strong>option will not appear. Only selecting a File/Asset will provide this option.</td></tr>
</tbody></table></figure>
<p>Expanding the <strong>Settings </strong>section will provide these options for your URL or Asset:</p>
<p><img src="https://support.optisigns.com/hc/article_attachments/42307250844819" width="325" height="199"></p>
<ul>
<li>
<strong>Duration (seconds) - </strong>When the link is clicked or tapped, how long the link or asset will display before returning to the main design.</li>
<li>
<strong>Preload</strong> <strong>- </strong>When checked, the website or image will load when the design does. This reduces the amount of time it takes for the website to pop up when the link is clicked. However, having numerous links needing to load may cause lag.</li>
</ul>
<p>Expanding the <strong>Close &amp; Nav Bar Styler </strong>section will provide these options for your URL or Asset:</p>
<p><img src="https://support.optisigns.com/hc/article_attachments/42307250845203" width="269" height="422"></p>
<p>This section refers to the button that will appear on your screens when it is pushed to them. Clicking or tapping this button will return you back to the original design.</p>
<p><img src="https://support.optisigns.com/hc/article_attachments/42307234479891" width="313" height="135"></p>
<ul>
<li>
<strong>Icon Position - </strong>Shows where this icon will appear. Defaults to<strong> </strong>Bottom Center, but can be placed anywhere along the top or bottom, or turned off entirely.</li>
<li>
<strong>Icon Style - </strong>Allows you to set the styling of the icon. Choose from Label, Solid, Round, or Default (see above)</li>
<li>
<strong>Icon Size - </strong>Allows you to change the icon’s size. Choose from Small, Medium, Large, or Default (see above)</li>
<li>
<strong>Icon Color </strong>- Change the hexadecimal HTML color of the icon to better fit your design.</li>
<li>
<strong>Background Color - </strong>Change the hexadecimal HTML color of the icon background to better fit your design.</li>
<li>
<strong>Opacity </strong>- Slider allows you to change how opaque or transparent the icon background is.</li>
<li>
<strong>Show Navigation Bar </strong>- When checked, adds small navigation arrows to the icon. Allows you to navigate quickly through multiple links. Looks like this:</li>
</ul>
<p><img src="https://support.optisigns.com/hc/article_attachments/42307234480787" width="253" height="105"></p>
