<h2 id="h_01JY52A05AYYC6AFRRCCYM2E65">Basics of the Brand Kit</h2>
<p>The <strong>Brand Kit </strong>option can be selected from the Designer Side Menu:</p>
<figure class="wysiwyg-image"><img style="aspect-ratio: 308/569;" src="https://support.optisigns.com/hc/article_attachments/42313846696339" width="308" height="569"></figure>
<p>The most critical option here is <strong>Enforce Branding</strong>. This is a toggle which switches between three modes:</p>
<ul>
<li>
<strong>None - </strong>Enforce Branding is not enabled.</li>
<li>
<strong>Guideline - </strong>Enforce Branding is enabled, but the preselected fonts and colors are only used as a guideline. Users will still be able to use any font or color.</li>
<li>
<strong>Enforce - </strong>Enforce Branding is strictly enabled. Any font or color outside of those preselected will be disabled for normal users.</li>
</ul>
<p>When <strong>Enforce Branding </strong>is enabled, new options become available in the Brand Kit side menu: <strong>Brand Fonts </strong>and <strong>Brand Colors</strong>.</p>
<h3 id="h_01JY52A05F1AHMF6NQSEC29920"><span style="color: #434343;">Fonts Management</span></h3>
<p>The <strong>Brand Fonts </strong>section can be used to manage accepted fonts through the Brand Kit. To add fonts from within OptiSigns, hit <strong>Add Brand Font</strong>.</p>
<figure class="wysiwyg-image"><img style="aspect-ratio: 337/225;" src="https://support.optisigns.com/hc/article_attachments/42313875422611" width="337" height="225"></figure>
<p>Numerous fonts can be selected from the list of <strong>180 </strong>fonts available on Designer:</p>
<figure class="wysiwyg-image"><img style="aspect-ratio: 603/619;" src="https://support.optisigns.com/hc/article_attachments/42313875422995" alt="firefox_9lRMDVaico.gif" width="603" height="619"></figure>
<p>Once fonts are selected, they’ll be visible under the <strong>Brand Fonts </strong>section.</p>
<figure class="wysiwyg-image"><img style="aspect-ratio: 333/257;" src="https://support.optisigns.com/hc/article_attachments/42313846699539" width="333" height="257"></figure>
<p>The yellow font signifies the Default font for all users. Hovering over another font lets you set it as Default or Remove it from the approved font list.</p>
<p>With Enforce Branding enabled, only fonts on this list will be available for users when they select a font from the Font picker.</p>
